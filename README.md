# google-antigravity-appimage
An AppImage package to be installed on linux atomic system like Bazzite, Aurora and Bluefin 

Google Chrome Integration:
1. Install Chrome using flatpack (or bazaar)
   `flatpak install flathub com.google.Chrome`

2. Go to settings engine on top right, click on Open Antigravity User Settings > Browser

3. Fill Chrome Binary Path with chrome bin.
   On Bazzite: `/var/lib/flatpak/app/com.google.Chrome/current/active/export/bin/com.google.Chrome`

Enjoy!
